# Asset Files

The app will use Expo's default placeholder assets during development.

For production, replace with proper PNG files:
- icon.png (1024x1024)
- splash.png (1242x2436)  
- adaptive-icon.png (1024x1024)
- favicon.png (48x48)

You can generate these using online tools like:
- https://www.appicon.co/
- https://icon.kitchen/
- Or design custom icons in Figma/Photoshop
